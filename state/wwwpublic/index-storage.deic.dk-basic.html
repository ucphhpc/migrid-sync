Getting Started

We've gathered information about DeiC Storage in a few short intro guides and a more thorough user guide

  • Sign Up Intro for UCPH and external users
  • Workgroup Intro about creating, managing and accessing Workgroups
  • Network Drive Intro about using DeiC Storage as a network drive on your computer (only covers Windows for now)
  • Seafile Sign Up and Sync Intro for UCPH and external users
  • DeiC Storage Overview if you're not sure whether to use DeiC Storage's main storage or Seafile
  • User Guide with general usage instructions

We strongly recommend reading and following at least the first intro guide if you want to use DeiC Storage.
