IDMC Tips

This is only a helper page which is used by the actual front page

New User Interface

Did you know that you can test-drive the new modernized IDMC user interface? You can easily switch between the traditional and the new web interface by changing User Interface from V2 to V3 or vice versa on your Settings.

2-Factor Authentication (2FA)

Do you worry about your IDMC account security? IDMC provides optional 2FA on all services to significantly reduce the risk of account abuse even if someone gets hold of your password. Due to general security considerations and in particular the recent increase in phishing attacks we recommend enabling it if you haven't already done so. It's done by going through the little 2-Factor Auth Setup wizard.

Custom Start Page

Do you mainly use e.g. Files when on IDMC web? In your Settings you can select another IDMC Default Page to always go directly there when you log in. Thus, saving a bit of time and effort choosing it from the default Home every time.

Basic Web Hosting and Publishing

Are you aware that IDMC integrates basic web hosting for Workgroups e.g. to present your research and projects? When you create a Workgroup on IDMC it automatically comes with a fully public and a participant-only web space where you can make your own web pages. Please refer to the user guide for further details.

Easy Data Exchange

Do you sometimes need to exchange files with people not on IDMC? You can easily do so by right clicking a folder in your IDMC Files and selecting Share Link and Create. Then choose Read Access and/or Write Access depending on whether you want the link recipient to download or upload files in the selected folder. Anyone you send or hand the link to can access the data without the need to have an IDMC account. Please refer to the user guide for further details.

IDMC as a Network Drive

Have you ever wanted to use your IDMC files seamlessly from your local programs? If you set up SFTP or WebDAVS access on your IDMC SFTP Setup or WebDAVS Setup you can in fact do just that. Depending on platform and personal preferences you can pick either of the two access methods and map/mount your IDMC home on your computer to interact with the files and folders there like any local data. Please refer to the user guide for further details.

Easy data Analysis and Presentation

Why worry about having the right tools for your data analysis and presentation on you own computer? In a lot of cases you can just use IDMC and the integrated Jupyter DAG/MODI services to do a many such tasks directly on your IDMC data, without the need to maintain special-purpose programs locally. In teaching or workshop situations this also comes in handy, because it removes the hassle for students/participants to install and setup needed applications in the required versions. With Jupyter they just need a computer and an IDMC account. Please refer to the user guide for further details.

Data Distribution with Import Sharelink

Ever needed to hand out a copy of some data to other IDMC users? A common way of distributing files e.g. for courses or workshops is by means of making one or more folders with material and creating read-only sharelinks on them. The participants can then select a folder on ther Files page and use Share Link -> Import from the right-click menu to easily fetch their own copy directly - without downloading to their own computer first. This is particularly convenient for Jupyter notebooks to be used in exercises.

External User Accounts with Peers

What if you want to invite externals onto IDMC for collaboration or courses? If you are employed at UCPH you can use the built-in Peers functionality to accept or invite people outside UCPH onto IDMC. That is, with their own personal account using a non-UCPH login. Click Add from your Home page and check the Peers app then Save to activate Peers in your app navigation menu. From there you can inspect and accept any existing external user account requests that specifically referenced you as contact on the Requested Peers tab. Additionally you can use Enter Peers or Import Peers to define and invite externals to sign up easily. For all such peers you define an Expire value, which is used by the administrators to time-limit peer account access and accept the mandatory annual account renewals in the given period. You can always extend it later if needed, however. Peers can similarly be used to preserve access for your former colleagues if you need it for continued collaboration.

YubiKey For Multifactor Authentication

Can I use a YubiKey security device for stronger and safer authentication? UCPH generally recommends securing your logins with MFA and offer employees e.g. the small keyring-format hardware security keys from Yubico for the purpose. You might want to have a word with your local leader about perhaps getting one if you haven't done so already. It can be used for various common UCPH services and we offer basic support for it on ERDA/IDMC and SIF, too. For now you can use it along with the Yubico Authenticator app to actually get a 3-factor authentication. Namely, your usual user credentials, your physical YubiKey and the generated TOTP 6-digit token from the associated app. All you need to do is to install the Yubico Authenticator app, scan your 2-factor QR code from your Setup page here in it and save with 'Require touch' enabled. Then you can get your 6-digit TOTP token from the app when hooked up with your YubiKey. Please refer to the vendor documentation for further instructions and details about the setup.
